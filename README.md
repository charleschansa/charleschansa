### ⚠️ Enable Snake Animation (Important)

To make the snake animation work on your profile:

1. Create a new repository named exactly: **charleschansa** (same as your username)
2. Add your README.md (this file) there
3. Create a folder: `.github/workflows/`
4. Inside it, create a file called: `snake.yml`
5. Paste this code:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: charleschansa
          outputs: |
            dist/github-contribution-grid-snake.svg
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

6. Commit & push
7. Wait a few minutes — the snake will start moving 🐍
